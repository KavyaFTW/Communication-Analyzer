# AI Chat Sentiment Analyzer

An NLP-based web application that analyzes conversations and determines sentiment, communication health, positive indicators, and negative indicators.

 Features

- Sentiment Analysis
- Communication Health Score
- Green Flag Detection
- Red Flag Detection
- Positive & Negative Word Extraction

 Tech Stack

- React.js
- Node.js
- Express.js
- Sentiment.js

How to Run

Backend

```bash
cd server
npm install
node index.js
```

Frontend

```bash
cd client
npm install
npm start
```

 Sample Output

- Positive 
- Neutral 
- Negative 
- Green Flags 
- Red Flags
